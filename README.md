# AuraHire

AuraHire is an AI-powered interview preparation platform that helps students prepare for technical interviews through resume analysis, AI mock interviews, and personalized feedback.

## Features

- AI Resume Analysis
- AI Mock Interviews
- Personalized Feedback
- Progress Dashboard
- User Authentication

## Tech Stack

- React
- JavaScript
- Vite
- Git
- GitHub

## Project Status

This project is currently under development.
